# Meteor-Twitter-Clone
A Simple Twitter Clone Made with MeteorJS and MongoDB

<h1>Installation</h1>
<ul>
<li>Download Repository</li>
<li>Install Meteor in the directory : npm install meteor</li>
<li>Run using : meteor
</ul>
